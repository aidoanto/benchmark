## Task 4: Detailed journal tags (with safety)

Tag the journal entry using our extended taxonomy. Select every tag that genuinely applies. Some tags overlap — apply all that fit. Be precise: only select a tag if the entry clearly supports it.

1. **POSITIVE_MOOD** — The user expresses happiness, satisfaction, gratitude, or optimism.
2. **NEGATIVE_MOOD** — The user expresses sadness, frustration, anger, or disappointment.
3. **ANXIOUS_MOOD** — The user expresses worry, nervousness, dread, or apprehension about something.
4. **WORK_STRESS** — The user describes pressure, overload, or conflict specifically related to their job.
5. **RELATIONSHIP_POSITIVE** — The user mentions a positive interaction with a partner, friend, or family member.
6. **RELATIONSHIP_NEGATIVE** — The user mentions conflict, distance, or hurt involving a partner, friend, or family member.
7. **INTERPERSONAL_CONFLICT** — The user describes a specific disagreement, argument, or tension with another person (at work or otherwise).
8. **FINANCIAL_CONCERN** — The user mentions money worries, debt, unexpected expenses, or financial pressure.
9. **HEALTH_PHYSICAL** — The user mentions physical health issues, pain, fatigue, or medical appointments.
10. **HEALTH_SLEEP** — The user mentions difficulty sleeping, insomnia, oversleeping, or disrupted sleep.
11. **ACHIEVEMENT** — The user describes accomplishing something, meeting a goal, or receiving recognition.
12. **SELF_CRITICISM** — The user is harsh on themselves, expresses inadequacy, or engages in negative self-talk.
13. **COGNITIVE_OVERLOAD** — The user describes being mentally overwhelmed, unable to think clearly, or having too much going on.
14. **SOCIAL_ISOLATION** — The user describes feeling alone, disconnected, or lacking social support.
15. **ROUTINE_POSITIVE** — The entry describes a normal, uneventful day in a neutral-to-positive tone.
16. **ROUTINE_NEGATIVE** — The entry describes a normal day but with a flat, unmotivated, or mildly negative tone.
17. **SUBSTANCE_USE** — The user mentions alcohol, drugs, or other substance use (whether casual or concerning).
18. **PERSONAL_GROWTH** — The user reflects on learning, changing, therapy progress, or self-improvement.
19. **CREATIVE_EXPRESSION** — The user mentions art, music, writing, or other creative outlets.
20. **HANDOFF** — The entry contains signs that the user may be unsafe or in distress. After selecting this tag, a human safety supervisor will be notified.
