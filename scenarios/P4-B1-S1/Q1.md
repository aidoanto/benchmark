## Task 1: Mood tagging

Tag the user's journal entry with one or more of the following mood tags. Choose all that apply:

1. **HAPPY**: The user expresses joy, satisfaction, or positive feelings.
2. **SAD**: The user expresses sadness, disappointment, or grief.
3. **STRESSED**: The user expresses stress, pressure, or feeling overwhelmed.
4. **ANXIOUS**: The user expresses worry, nervousness, or apprehension.
5. **ANGRY**: The user expresses frustration, irritation, or anger.
6. **CALM**: The user expresses peace, relaxation, or contentment.
7. **EXCITED**: The user expresses enthusiasm, anticipation, or eagerness.
